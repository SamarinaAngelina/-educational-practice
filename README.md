## 🛠 Инструкция по локальному запуску

### 1. Запуск Сервера
1. Откройте терминал и перейдите в папку сервера: `cd server`
2. Установите необходимые библиотеки: `npm install`
3. Запустите сервер: `npm start`
После этого сервер начнет работу на адресе `http://localhost:3000`.

### 2. Запуск Клиента
1. Перейдите в папку `client`.
2. Дважды кликните на файл `index.html`, чтобы открыть его в любом браузере.
